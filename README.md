# SwissMilk
SwissMilk - FIDE Specification compatible Dutch pairing system library and a tournament manager system (WIP)

### Building
```bash
make PROFILE=release
make test PROFILE=release
./tests/run_tests
```

For debug builds
```bash
make
make test
./tests/run_tests
```
